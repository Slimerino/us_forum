How to install

  1. Add everything in /usersc/ to your usersc
  2. Add /forums/ into your main directory
  3. Upload us_forum.sql to your database (The same db as your UserSpice db)
  4. Follow below on how to setup your init
  5. Open init.php and copy that to your users/init.php and setup the details it should look like the image below (code is in init.php code.txt)

Also in /forums/view-thread.php find "if(hasPerm([2],$user->data()->id)){" and edit the "2" to the permission levels you want to have a moderation panel (For multiple do 2,3,4 etc)
